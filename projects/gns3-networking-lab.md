# GNS3 Networking Lab

## Overview

This project was created to practice hands-on networking skills in a virtual lab environment using GNS3 and VirtualBox. The lab included multiple devices and operating systems to simulate realistic connectivity, routing, and troubleshooting scenarios.

## Objectives

- Build a working virtual network topology
- Configure connectivity between devices
- Practice troubleshooting routing and network communication
- Strengthen understanding of virtualization and network services
