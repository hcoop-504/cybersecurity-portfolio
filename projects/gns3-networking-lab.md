# GNS3 Networking Lab

## Overview

This project highlights hands-on networking labs I built using GNS3 and virtual machines to strengthen my understanding of network configuration, connectivity, and troubleshooting in a simulated environment. The labs allowed me to work with routers, switches, Linux systems, and security-focused virtual machines while practicing how devices communicate across a network. Through these projects, I developed experience building topologies, testing connectivity, and troubleshooting common issues in a controlled lab setting.

## Objectives

- Build and manage virtual network topologies in GNS3
- Practice router and switch configuration in a lab environment
- Connect virtual machines across multiple network segments
- Strengthen troubleshooting skills using networking tools and command-line utilities
- Improve understanding of routing, NAT, DHCP, and lab design concepts

## Lab Environment

The networking labs were built using a combination of GNS3 and virtualization platforms. Depending on the lab, the environment included:

- GNS3 for topology design and network simulation
- VirtualBox for running virtual machines
- Linux-based systems such as Ubuntu, Kali Linux, and Tiny Core Linux
- Virtual routers and switches
- Firewall and NAT configurations for internet-connected lab scenarios

## Key Activities Completed

Across these labs, I completed tasks such as:

- Built multi-device network topologies in GNS3
- Connected routers, switches, and Linux virtual machines in the same lab
- Configured and tested IP addressing and interface connectivity
- Practiced troubleshooting communication issues between hosts
- Worked with routing concepts to allow traffic to move between segments
- Used NAT in lab environments to provide controlled internet access
- Explored DHCP and other network services in virtual lab scenarios
- Verified connectivity with tools such as `ping`, `traceroute`, and Linux network commands

## Skills Demonstrated

This project helped me develop and demonstrate skills in:

- Network topology design
- Router and switch fundamentals
- IP addressing and connectivity testing
- Routing and NAT concepts
- Linux networking
- Troubleshooting and problem solving
- Virtual lab deployment and management
- Technical documentation
